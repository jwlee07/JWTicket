# JWTicket

## 세팅 순서

### VSCODE 설치 및 초기 세팅
1. vscode 설치
2. Python Extension Pack 설치

### 가상환경 세팅
1. python -m venv venv 
2. source venv/bin/active

### Django 설치
1. pip install django
2. pip install django-lint

### 웹 크롤링 라이브러리 설치
1. pip install requests
2. pip install bs4
3. pip install selenium
